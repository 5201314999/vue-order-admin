# order-admin

> 用 Vue 全家桶搭建的商品订单管理系统

## 预览项目
1. 
```bash
npm install
npm start
```
2. 测试账号： test/123

## 技术栈
- Vue 2.0
- vue-router
- Vuex
- Axios
- Element-UI
- ES6 && ES7
- Sass
